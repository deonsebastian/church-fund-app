# Church Fund App

## About This Project

Church Fund App is a simple web application made to help manage church funds and related records. It is built as a frontend web app and can be opened in any browser.

This project is already built and deployed online.

---

## Live Website

Open the app here:
https://churchfundapp.vercel.app/

---

## Main Purpose

This app helps to:

* Manage church fund data
* View records easily
* Keep information organized

---

## Built With

* HTML
* CSS
* JavaScript
* Vite build tool
* Tailwind CSS

---

## Project Folder Structure

```
church-fund-app-gh-pages/
│
├── index.html

```

---

## How To Run Locally

1. Download or clone the project
2. Open the folder
3. Double-click **index.html**

The app will open in your browser.

---

## Deployment

This project is deployed using Vercel.
The production files are inside the **gh-pages build folder**.

---

## Notes

* This is a static build version.
* No installation is needed to run it.
* Works best in modern browsers.
